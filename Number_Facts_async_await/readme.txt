Project 1: Number Facts
Description
This project allows users to fetch interesting facts about numbers. It showcases the use of the Numbers API to retrieve facts for a single number, multiple numbers, or get multiple facts about a single number.

How to Run
Ensure you have a modern browser installed.
Open the index.html file in your browser.
Follow the on-screen instructions to get facts about numbers.
Features
Fetch a fact about a single number.
Get facts about multiple numbers in a single request.
Get four facts about a single number.
Technologies Used
JavaScript
HTML
CSS
Numbers API
