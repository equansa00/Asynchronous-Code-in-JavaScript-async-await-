Pokémon Facts Project
Description:
This project lets users explore random Pokémon by fetching data from the Pokémon API. Upon request, it showcases random Pokémon along with their image and provides a detailed description of the species.

How to Run:
Ensure you have a modern browser installed.
Navigate to the project directory and open the index.html file in your browser.
Use the "Fetch Random Pokemon Data" button to display three random Pokémon on the page, along with their images.
If you'd like to fetch descriptions of the displayed Pokémon species, simply use the corresponding button to add this information to the display.
Features:
Random Pokémon Fetch: Fetch and display three random Pokémon every time you click the button. This showcases the Pokémon's name along with their image.
Species Description: Delve deeper and fetch a description of the Pokémon species to learn more about them.
Technologies Used:
JavaScript: For fetching and processing data from the Pokémon API.
HTML: For structuring the web page.
CSS: Can be added to style the webpage according to preference.
Pokémon API: An external API that provides data about Pokémon.
Notes:
The project leverages promises to handle asynchronous operations and ensure smooth data flow.
The script.js file contains the primary logic for fetching data and updating the UI.
