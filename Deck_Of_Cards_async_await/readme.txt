Project 2: Deck of Cards
Description
This project is a fun and interactive way to draw playing cards using the Deck of Cards API. Users can draw a single card or multiple cards from a deck and see the card's image, value, and suit.

How to Run
Ensure you have a modern browser installed.
Open the index.html file in your browser.
Click the on-screen button to draw a card or multiple cards from a deck.
Features
Draw a single card from a newly shuffled deck and view its image.
Draw two cards from the same deck in successive requests.
Display the card's image, value, and suit.
Technologies Used
JavaScript
HTML
CSS
Deck of Cards API
